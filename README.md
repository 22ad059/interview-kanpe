# 🎤 面接カンペ

Zoom などのオンライン面接中に、カンペ（メモ）を**常に最前面の小窓**に表示できるWebアプリです。

## 使い方

1. Chrome または Edge でアプリを開く
2. カードにカンペを書く（自動保存されます）
3. 「📌 カンペを最前面に表示」を押すと小窓が開き、Zoomを全画面にしても常に手前に浮かびます

## 機能

- カード形式でカンペを整理（追加・削除・並べ替え）
- 内容はブラウザ（localStorage）に自動保存 — サーバーには何も送信されません
- 最前面小窓：カードへのジャンプボタン、文字サイズ調整、編集のリアルタイム反映

## 注意

- 最前面表示は Document Picture-in-Picture API を使用しているため、**Chrome / Edge が必要**です
- Zoomで画面共有するときは「画面全体」ではなく**特定のウィンドウだけ**を共有すれば、カンペは相手に見えません
- カンペは読み上げず、ヒントとして使うのがおすすめです
